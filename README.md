# dotfiles

```sh
dotfiles/
├─ git/
│  ├─ profiles/
│  │  ├─ gitconfig-private.conf
│  │  ├─ gitconfig-work.conf
│  ├─ .gitconfig
│  ├─ .gitignore
├─ scripts/
│  ├─ git-clean-branches.sh
├─ zsh/
│  ├─ .zshalias
│  ├─ .zshrc
├─ README.md
```
